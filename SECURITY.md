# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in runsheet, please report it
responsibly via [GitHub Security Advisories] rather than opening a public issue.

1. Go to the [Security tab] of the repository
2. Click **Report a vulnerability**
3. Provide a description and steps to reproduce

You should receive an initial response within 72 hours. We'll work with you to
understand the issue, confirm the fix, and coordinate disclosure.

## Supported versions

Only the latest published version receives security updates.
